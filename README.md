# homebrew-journal

Homebrew tap for [Journal](https://github.com/jamiefdhurst/journal) — a simple personal journal web application written in Go.

## Install

```bash
brew tap jamiefdhurst/journal
brew install journal
```

## Upgrade

```bash
brew update
brew upgrade journal
```

## Uninstall

```bash
brew uninstall journal
brew untap jamiefdhurst/journal
```
